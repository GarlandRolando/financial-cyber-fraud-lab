# Upload steps

## Repository 1: profile

1. Create a public repository named exactly `GarlandRolando`.
2. Upload the files from the `GarlandRolando` folder.
3. Commit with `Create financial risk technology profile`.

## Repository 2: project

1. Create a public repository named `financial-cyber-fraud-lab`.
2. Do not initialise it with GitHub's generated files.
3. Upload the entire project folder.
4. Commit with `Initial financial cyber-fraud analytics project`.
5. Open Actions and verify that CI passes.
6. Add topics:
   - financial-crime
   - fraud-analytics
   - cybersecurity
   - banking
   - python
   - sql
   - regtech
7. Pin the repository to your profile.

## Before applying to jobs

- Run the Streamlit app.
- Read the model card.
- Change at least one feature or threshold.
- Add one chart or investigation workflow yourself.
- Be able to explain false positives and false negatives.
